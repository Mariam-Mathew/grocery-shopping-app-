# Initial Setup Prompts

## Project Creation
```
Create a new React Native Expo project with TypeScript and tab navigation. Include:
- Expo Router for navigation
- TypeScript configuration
- Basic tab structure (Home, Cart, Settings)
- Initial styling setup
- Package.json with necessary dependencies
```

## Dependencies Installation
```
Install and configure the following packages:
- expo-router for navigation
- @types/react-native for TypeScript
- react-native-screens for navigation
- react-native-safe-area-context for safe areas
- Any additional UI libraries needed
```

## Basic Configuration
```
Set up the following configurations:
- TypeScript (tsconfig.json)
- Expo Router configuration
- Metro bundler configuration
- Babel configuration
- ESLint and Prettier if needed
```

## Initial File Structure
```
Create the basic file structure:
- app/ directory with router files
- components/ directory for reusable components
- constants/ directory for app constants
- assets/ directory for images and fonts
- types/ directory for TypeScript definitions
```

## First Commit
```
Initialize Git repository and create initial commit with:
- Basic project structure
- Configuration files
- Package.json and dependencies
- README.md with project description
```
