# Testing and Deployment Prompts

## Unit Testing Setup
```
Set up comprehensive unit testing with:
- Jest and React Native Testing Library
- Test utilities for components
- Mock implementations for context providers
- Snapshot testing for UI components
- Test coverage reporting
- Automated test running in CI/CD
```

## Component Testing
```
Write tests for all major components:
- ProductCard component rendering and interactions
- CartItem component with quantity controls
- SearchBar component with input handling
- FilterModal component with filter application
- Theme toggle functionality
- Navigation between tabs
- Modal open/close behavior
```

## Integration Testing
```
Create integration tests for:
- Add to cart flow from product list
- Cart total calculation accuracy
- Theme persistence across app restarts
- Search and filter combination
- Settings preference changes
- Deep linking navigation
- AsyncStorage operations
```

## End-to-End Testing

- Accessibility testing automation
